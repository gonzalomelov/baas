The best and easiest way to develop your application
UBaaS provides a robust backend for your mobile apps by providing features that enable you to build amazing apps without worrying about your backend. Currently UBaaS provides the following services:
  • AppData A query-able key/value (and more) data storage platform for your app
  • Users Keep track of users of your application and control access to data on a user-by-user basis
  • Access control Control the access to the data via settings roles with different permission at entity’s collection level.
  • Push Notifications Our library uses GCM integration which gives you awesome push features without the headache of managing them yourself.
  • Offline AppData Allows your app to keep working without internet connection.
  • Usage Statistics Provides real-time usage statistics of your application

About

UBaaS is a Backend as a Service provider that makes it easy for developers to setup, use and operate a cloud backend for their mobile apps. They don't have to worry about connecting to various cloud services, setting up servers for their backend, or maintaining and scaling them.

With UBaaS, developers simply code their app using Android’s SDK or REST API. They then connect the app to UBaaS's backend features using these services, and the app is ready to launch. We'll worry about auto-scaling and maintaining your backend. You focus solely on your app.
